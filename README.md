# OM MURUGA BATTERIES & INVERTERS

A modern, high-performance web application built for a battery and power solutions retailer. This project focuses on delivering a premium user experience, rapid performance, and a seamless lead-generation inquiry flow.

## Features

- **Premium UI/UX:** Built with a fully responsive layout featuring smooth scrolling, interactive hover animations, and a polished design system.
- **Dynamic Store Hours:** A real-time status indicator dynamically displays "Open Now" or "Closed" based on the user's current time and the store's custom business hours.
- **Product Inquiry Cart:** A streamlined cart-like system where users can build a list of products they are interested in.
- **WhatsApp Integration:** A fully integrated WhatsApp direct-messaging flow. Users submit their inquiries via a modal or drawer, and the system automatically generates a formatted, friendly message directly opening in the WhatsApp client.
- **Regional Testimonials:** A custom, auto-scrolling testimonial carousel featuring multilingual support for enhanced local trust.

## Tech Stack

- **Frontend Framework:** React.js (Vite)
- **Styling:** Vanilla CSS with comprehensive media queries for mobile-first responsiveness
- **Icons:** Lucide React & Custom SVGs

## Getting Started

To run this project locally:

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm run dev` to start the local Vite development server
4. Build for production using `npm run build`

## Deployment

This project is fully optimized and ready to be deployed to Vercel or any standard static hosting provider.
